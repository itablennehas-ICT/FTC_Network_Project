# FTC Network Project
> Multi-floor enterprise network | Cisco Packet Tracer | March 2026

## 📌 Project Overview
This project is an enterprise network simulation for FTC CLUB using Cisco Packet Tracer.  
It demonstrates the design and implementation of a structured network including VLANs, routing, and security configurations.

## Objectives
- Design a scalable and organized network
- Implement VLAN segmentation
- Configure routing between networks
- Ensure basic network security
- Simulate real enterprise environment

##  Network Features
- VLAN configuration
- Inter-VLAN routing
- DHCP configuration
- NAT (Network Address Translation)
- ACL (Access Control Lists)
- Switch and Router configuration
- IP addressing plan
